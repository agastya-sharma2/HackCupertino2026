# Fitness Dashboard

A professional fitness tracking dashboard built with Flask.

## Features

- **Diet Tracking**: Log meals with protein, carbs, and quantities. Account for allergies.
- **Hydration**: Track water intake with goals, adjusted for activity and temperature.
- **Workout Plans**: Log workouts, get rest plans based on goals.
- **Game Analysis**: Statistical analysis of sports performance.
- **Gamification**: Earn points for workouts and games to unlock cheat meals.
- **Pre/Post Workout Guides**: Tips for preparation and recovery.

## Setup

1. Install dependencies:
   ```
   pip install flask
   ```

2. Run the app:
   ```
   python app.py
   ```

3. Open http://127.0.0.1:5000/ in your browser.
