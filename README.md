# HackCupertino2026
&lt;hack> Cupertino 2026 hackathon

## Inspiration
We saw that most fitness trackers only help track and help you during your workout, however, there is nothing that helps somebody to prepare for their workout and recover from their workout.
## What it does
Just give the website details about your situation (cardio workout, agility training, etc.) and some of your health details and it'll give you a guide on what to eat, how much to eat, warmups, stretches, main workouts, and recovery. Everything from start to finish that you need for a proper work out.
## How we built it
We used the Python module Flask to build the website and we used Openrouter for the AI models by using their APIs. We also used Html, CSS, and Javascript.
## Challenges we ran into
It was hard to visualize our final goal so there was a lot of disagreements. Finding a reliable AI provider was also a challenging journey. The code was also getting tricky to navigate in. Also, this code might not be fully working with the AI since the API keys are old.
## Accomplishments that we're proud of
Despite the disagreements, we pulled through and have a fully working product to show.
## What we learned
We learned how to organize code, use GitHub, integrate AI APIs into a project, and how to prepare for a workout.
